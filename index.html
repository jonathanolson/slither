What is Slitherlink?

Slitherlink is a puzzle where you need to draw a single continuous loop on a planar graph that has:

  • Vertices: Dots/points where the loop can turn.
  • Edges: Connections between vertices that are either part of the loop or not.
  • Faces: Regions surrounded by edges that are either inside or outside the loop. Some faces contain a number.

The loop must satisfy the following rules:

  • The loop cannot cross or touch itself.
  • Each face with a number must have exactly that many edges of the loop around it.

Furthermore, valid puzzles will have a single possible solution. This enables a class of techniques called highlander rules ("there can be only one").

Note: the terminology I'm using is a bit more mathematically skewed than the typical Slitherlink terminology.
