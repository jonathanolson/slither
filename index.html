What is Slitherlink?

Slitherlink is a logic puzzle where you draw a single continuous loop on a grid. The goal is to connect dots without crossing or overlapping, creating a loop that adheres to specific rules based on the numbers in some cells.

I'll use the following mathematics-oriented terminology to describe things:

  • Vertices: Dots where the loop can turn.
  • Edges: Lines between vertices that may be part of the loop.
  • Faces: Areas surrounded by edges, with some faces containing numbers indicating the number of edges that must surround them.

The loop must satisfy the following rules:

  • The loop cannot cross or touch itself.
  • Each numbered face must have exactly that many edges of the loop around it.

Valid puzzles have a single solution, allowing the use of highlander rules ("there can be only one").

Puzzles are a configuration of numbers on a board (originally a grid of squares, but it can be any planar graph).

Note: the terminology I'm using is a bit more mathematically skewed than the typical Slitherlink terminology.

How is Slitherlink Typically Solved?

If you're new to Slitherlink, the Wikipedia article is a great starting point. There are also excellent resources and guides available online.

Typically, solvers learn to recognize specific patterns of numbers, lines, and x's that allow them to deduce where other lines or x's must go.
